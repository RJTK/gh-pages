Duality for Semidefinite Programming

The idea of duality is of fundamental importance in convex optimization. The procedure to obtain a dual optimization problem \((\mathcal{D})\) from the primal problem \((\mathcal{P})\) is similar to the procedure we followed to obtain \((\mathcal{P})\) in the first place; namely, we consider a family of functions \(g(\Lambda)\) which are relaxations of \((\mathcal{P})\) in the sense that \(g(\Lambda) \le p^\star\) for every parameter \(\Lambda\) where \(p^\star\) is the optimal objective value of problem \((\mathcal{P})\), and then we maximize this inequality over \(\Lambda\) to obtain the greatest lower bound.

To find the function \(g\), we begin with the Lagrangian \(\mathcal{L}(P, \Lambda)\), as is likely familiar to most readers:

\begin{equation} \begin{aligned} \mathcal{L}(P, \Lambda) &= x_0^{\mathsf{T}} P x_0 + \langle \Lambda, A^{\mathsf{T}} P A - P + Q\rangle\\ \implies g(\Lambda) &\overset{(\text{def})}{=} \underset{P \succeq 0}{\text{inf}}\ \mathcal{L}(P, \Lambda), \end{aligned}\notag \end{equation}

where the inner product is \(\langle C, Z \rangle\) is furnished by the trace: \(\mathsf{tr}(C^{\mathsf{T}} Z)\).

As long as we restrict the domain of \(g\) to be \(\Lambda \succeq 0\), then this is a suitable definition of the relaxation function \(g\), since for any \(P \succeq 0\) which is feasible for \((\mathcal{P})\), the inner product \(\langle \Lambda, A^{\mathsf{T}} P A - P + Q\rangle \le 0\) and therefore \(g(\Lambda) \le p^\star\) for any \(\Lambda \succeq 0\).

By manipulating the inner product, particularly using the fact that \(\langle C, BZ\rangle = \langle B^{\mathsf{T}} C, Z \rangle\) we can turn this optimization problem into another semidefinite program. To do so, we first have:

\begin{equation} \begin{aligned} \mathcal{L}(P, \Lambda) &= x_0^{\mathsf{T}} P x_0 + \langle \Lambda, A^{\mathsf{T}} P A - P + Q\rangle\\ &= \langle\Lambda, Q\rangle + \langle X_0 + A \Lambda A^{\mathsf{T}} - \Lambda, P\rangle, \end{aligned}\notag \end{equation}

where \(X_0 = x_0 x_0^{\mathsf{T}}\). Now, in order for \(g(\Lambda) > -\infty\) (remember we are minimizing over \(P \succeq 0\)) we need the constraint \(A\Lambda A^{\mathsf{T}} - \Lambda \succeq -X_0\), in which case an optimizing value of \(P\) is simply \(0\). Thus, the problem of maximizing \(g\) becomes the SDP:

\begin{equation} \begin{aligned} \underset{\Lambda}{\text{maximize}}&\quad \langle\Lambda, Q\rangle\\ \text{subject to} &\quad \Lambda \succeq 0\\ &\quad A \Lambda A^{\mathsf{T}} - \Lambda \succeq -X_0, \end{aligned}\tag{$\mathcal{D}$} \end{equation}

and this is called the dual SDP of \((\mathcal{P})\). If we use \(d^\star\) to denote the value of \((\mathcal{D})\), then we have already established that \(p^\star \ge d^\star\). And, it is “usually” the case that strong duality holds in convex optimization: \(p^\star = d^\star\). For simplicity, I will simply assume that this equality holds.

A Dual Dynamical System

The dual SDP \((\mathcal{D})\) bears quite a resemblance to the primal problem \((\mathcal{P})\). Indeed, there appears the expression \(A^{\mathsf{T}} \Lambda A - \Lambda\), which looks exactly like the expression that would arise in the computation of a Lyapunov function for a linear system \[y(t + 1) = A^{\mathsf{T}} y(t).\] Indeed, because of this similarity, it is natural to refer to the system defining \(y(t)\) to be a dual system associated to \(x(t + 1) = Ax(t)\).

To obtain some insightful dual results, let’s look at another quadratic functional \(\Gamma(y_0)\), namely

\[ \Gamma(y_0) = \sum_{t = 0}^\infty y(t)^{\mathsf{T}} Z y(t), \]

for some matrix \(Z \succeq 0\).

Since we’re after a dual analysis, let’s try to find a lower-bound on this functional (rather than an upper bound, as in the primal case). To this end, consider a quadratic function \(\wedge(y) = y^{\mathsf{T}} \Lambda y\) (with \(\Lambda \succeq 0\)) as before, but now let’s suppose we can verify the inequality

\[ \wedge(A^{\mathsf{T}} y) - \wedge(y) \ge y^{\mathsf{T}} X y. \]

Similarly as before, we can obtain an inequality on \(\Gamma(y_0)\) by means of a telescoping summation: \(\wedge(y_0) \le \Gamma(y_0)\), which now makes for a lower bound. Finding the best lower bound entails maximizing \(\wedge(y_0)\) with respect to \(\Lambda\). This results in a semidefinite programming problem

\begin{equation} \begin{aligned} \underset{\Lambda}{\text{maximize}}&\quad y_0^{\mathsf{T}} \Lambda y_0\\ \text{subject to} &\quad \Lambda \succeq 0\\ &\quad A \Lambda A^{\mathsf{T}} - \Lambda \succeq -Z. \end{aligned}\tag{$\mathcal{D}$} \end{equation}

This takes the same form as the dual SDP where \(Q = y_0 y_0^{\mathsf{T}}\) and \(Z = x_0 x_0^{\mathsf{T}}\). That is, the constraint of the primal system is derived from the primal cost functional \(J\), which is then related to the initial point of the dual linear system \(y(t)\). Likewise, the initial point of the primal system, \(x_0\), can be related back to the dual quadratic functional \(\Gamma\) by the choice \(Z = x_0 x_0^{\mathsf{T}}\).

Remark: The particularly special form $Z = x_0 x_0^{\mathsf{T}}$ can be generalized for any $Z \succeq 0.$ To do so, one must be aware of the correspondance between positive semidefinite matrices and covariance matrices of random vectors. Following similar calculations as I’ve done here, the results are generalized by considering /random/ starting initial points $x_0, y_0$ for the linear systems where they have zero mean $\mathbb{E}[x_0] = \mathbb{E}[y_0] = 0$ and variances $\mathbb{E}[x_0 x_0^{\mathsf{T}}] = Z$, $\mathbb{E}[y_0 y_0^{\mathsf{T}}] = Q$.

Establishing Equality

Finally, using the strong duality between \((\mathcal{D})\) and \((\mathcal{P})\), i.e., that their values are equal: \(V(x_0) = x_0^{\mathsf{T}} P x_0 = y_0^{\mathsf{T}} \Lambda y_0 = \wedge(y_0)\), and the fact that we have the inequalities \(V(x_0) \ge J(x_0)\) and \(\wedge(y_0) \le \Gamma (y_0)\) it must follow that \(J(x_0) \le V(x_0) \le \Gamma (y_0)\) and \(J(x_0) \le \wedge(y_0) \le \Gamma(y_0)\). Therefore, if we can finally also show that \(J(x_0) = \wedge(y_0)\) (itself a rather neat finding!) then it must be the case that \(V(x_0) = J(x_0)\) and $∧(y_0) = Γ(y_0).$ That is, the bounds obtained from semidefinite programming are in fact equalities.

To show this equivalence is a matter of calculating and applying the definitions, keeping in mind that we’re working in the special case that \(Q = y_0 y_0^{\mathsf{T}}\) and \(Z = x_0 x_0^{\mathsf{T}}\) (though see the above remark – the result can be generalized by considering random initial conditions).

\begin{equation} \begin{aligned} J(x_0) &= \sum_{t = 0}^\infty \bigl(x(t)^{\mathsf{T}} y_0 \bigr)^2\\ &= \sum_{t = 0}^\infty \bigl((A^t x_0)^{\mathsf{T}} y_0 \big)^2\\ &= \sum_{t = 0}^\infty \bigl(x_0^{\mathsf{T}} (A^t)^{\mathsf{T}} y_0 \big)^2\\ &= \sum_{t = 0}^\infty \bigl(x_0^{\mathsf{T}} y(t) \big)^2\\ &= \Gamma(y_0). \end{aligned}\notag \end{equation}

Quadratic Functionals

In many applications (usually in problems of optimal control) we have associated with the system a cost functional \(J(x_0) = \sum_{t = 0}^T x(t)^{\mathsf{T}} Q x(t)\) where \(Q \succ 0\) is some given matrix and \(x(t + 1) = Ax(t)\) with \(x(0) = x_0\) defines the linear system with an initial point. This is a called a functional since it maps an entire trajectory of the linear system \(x(t)\) (starting at \(x_0\)) into \(\mathbb{R}\); functions which map from a vector space into the underlying scalar field are called “functionals”, rather than using the more generic term “function”.

While there are numerous methods to evaluate such functionals, it is interesting that they can be evaluated by means of semidefinite programming. This method of evaluation is by no means the best, but exploring some natural aspects of this approach is insightful and fascinating.

To see how this is possible, consider the following:

\begin{equation} \begin{aligned} J(Ax) - J(x) &= \sum_{t = 0}^\infty x(t)^{\mathsf{T}} A^{\mathsf{T}} Q Ax(t) - \sum_{t = 0}^\infty x(t)^{\mathsf{T}}Q x(t)\\ &\overset{(a)}{=} \sum_{t = 0}^\infty x(t + 1)^{\mathsf{T}} Q x(t + 1) - \sum_{t = 0}^\infty x(t)^{\mathsf{T}}Q x(t)\\ &= \sum_{t = 1}^\infty x(t)^{\mathsf{T}} Q x(t) - \sum_{t = 0}^\infty x(t)^{\mathsf{T}}Q x(t)\\ &= -x_0^{\mathsf{T}}Q x_0, \end{aligned}\notag \end{equation}

where \((a)\) follows since \(x(t + 1) = Ax(t)\) is a trajectory of the system. Following this calculation, suppose that we can find a function \(V(x) = x^{\mathsf{T}} P x\) such that \(V(Ax) - V(x) \preceq -x^{\mathsf{T}} Q x\). Then, by a telescoping summation we must have that \(V(x_0) \ge \sum_{t = 0}^\infty x(t)^{\mathsf{T}} Q x(t) = J(x_0)\). (If one wanted to be rigorous in these calculations, the possibility that one or both of these terms is \(\infty\) would need to be dealt with carefully). In any case, if we have such a \(V\), we also have an upper bound on the value of the quadratic functional. We would like for any such \(V\) to also be a Lyapunov function, so we will additional enforce the constraint that \(P \succeq 0\).

To find the best (i.e., least) upper bound for the given initial point \(x_0\) we can solve the semidefinite programming problem

\begin{equation} \begin{aligned} \underset{P}{\text{minimize}}&\quad x_0^{\mathsf{T}} P x_0\\ \text{subject to} &\quad P \succeq 0\\ &\quad A^{\mathsf{T}} P A - P \preceq -Q. \end{aligned}\tag{$\mathcal{P}$} \end{equation}

A natural question to ask is whether the bound \(V(x_0) \ge J(x_0)\) is actually an equality \(V(x_0) = J(x_0)\). To answer this question, we need to take a detour into the duality theory for semidefinite programming.
